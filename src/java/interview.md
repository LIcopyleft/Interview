### 基础知识点理解

- `rpc` 远程过程调用

-  `URI`（统一资源定位符）

- `RESTful`:Representational State Transfer ,REST的名称"表现层状态转化"中，省略了主语。"表现层"其实指的是"资源"（Resources）的"表现层"。

- 总结一下什么是RESTful架构：

        1.每一个URI代表一种资源；

        2.客户端和服务器之间，传递这种资源的某种表现层；

        3.客户端通过四个HTTP动词，对服务器端资源进行操作，实现"表现层状态转化"。
